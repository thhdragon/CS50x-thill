Plays on flag click. A short and awkward conversation between the scratch cat with a divers helmet on and a clown fish. I used variables as flags to time the chat. A rocket ship flies by and off the screen when the cat and fish variables = 1. I marked the ship animation done with a variable too for future additions. I wanted to try the broadcast message block so the final chat bubble from scratch cat is activated by a broadcast message block at the end of the rockets block. The rocket uses a custom block and is using the pick random block to randomize the color of the ship.
